今日份感想

发表于 2019-03-05 更新于 2019-03-07 分类于 生活 阅读次数：

2019.3.5距离毕业还有四个月的样子。但是感觉糟糕透了，无业游民的感慨吧。

2019.3.5离开了第二个实习公司，从去年11月到今年1月底，大概三个月的样子。最后工资没有达到预期范围离开了，也很感谢这三个月的时间，自己的进步也是很大的。到了这个公司接触了react/小程序，虽然都还在入门阶段，但是至少入门了。

在这三个月期间，模仿了一个豆瓣小程序的实现，使用Charles抓取接口，自己模仿ui。花了两周的时间，学习小程序，编写了原生小程序代码，基于Taro的小程序代码。同时也接触公司的业务，协助师傅完成一些h5页面的开发，后台管理系统的开发。整个实习过程中，任务是不多的，更多的是自己的学习。对于ES6的使用也有了新的认识，React的基本使用也能够熟练掌握。

辞职了，也就成为了一个无业游民，加入了春招大部队。秋招的时候，一直是对自己处于一个否定状态，面了几家没有结果之后，直接就放弃了。去了后面这个公司实习，这一次春招也是特别困难的，目前来说整个互联网行业都是比较惨的，各大公司裁员，春招压力无形来说是巨大的。没能顺利的走过秋招，希望能够自信的度过春招。

最后的大学时光，好好做毕设，找个时间出去旅游。也希望着能在最后的大学时间学习一些新的知识，希望能够在离开工大之前，学一点nodeJS和web前端安全性相关的东西。
