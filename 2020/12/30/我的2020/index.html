你好，2021

发表于 2020-12-30 更新于 2020-12-31 分类于 生活 阅读次数：

距离 2021 还有两天，好好的为今年写一份总结，流水账警告。

现在是 2020-12-31，写完这总结，估计距离今年结束只有 6 个小时。2020 是本命年，对我也像往年一样，不好不坏，并未出现本命年的某些魔怔，平平淡淡。

回顾

在去年的总结中，立了很多的 flag。不看不知道，一看吓一跳的效果。原来年初的我，对自己有这么多的期待。仔细回想，七成的 flag 都没拔掉，行动力和执行力非常低下。

  • Webpack/React/Node。是谁给我的勇气写插旗的
  • 把网络相关的知识复习一下。看了就忘是我本人
  • 坚持每天回家都刷题。完成 5 成吧，对某些数据结构/算法有了进一步理解
  • 读 12 本与工作无关的书籍。看了三本小说
  • 好好学习英语。我还是那个咸鱼
  • 能学会更多的菜品。红烧一切
  • 学一点理财的知识。我只追高，但是绝不杀跌
  • 用相机去记录杭州四季。我是疯了吧，写这个玩意

今年工作不是很忙，有了更多的时间学习。在博客中有记载的，其实涉及到的范围比较窄。花了很多时间重温数据结构和算法，也感觉到了进步；计算机网络也看了一次，但看完就忘记，可能是没找到正确的姿势学习；相关的前端知识，也没有太多的去研究，也优化了之前的一个 Music APP；学习了如何封装公用组件，但还没有用到实际；看了 react 的逐步实现，还学完系列。

工作

可能今年都是没有资格来说工作这件事情，去年毕业然后被裁员，今年进入了一家外企，过上了朝九晚五的生活，每天都很有规律且有很多自己的时间，但是也明显感觉到了与同行的差距。技术的老旧，而且工作量相对较少，工作内容/工作方式也和以前大不相同，所以以前的技术栈很难运用到现在的工作中。

由于是外企的原因，除了中国团队之外沟通都是靠英文，确实算是一个比较大的困难，虽然现在也很难听懂英文会议，但是邮件沟通越来越流畅了。

今年年初发生了疫情，公司出于安全的考虑，一直都没有全面复工，采取轮班制度，因此今年在家办公和去公司的时间对半分。在家的工作效率也是杠杠的！

在外企，怎么说呢。在每个人眼中都是工作相对轻松，工资相对来说过得去，有很多自己的生活，work life balance。目前的工作并不会带给人太多成就感，以前的我一直很抗拒这样的，总觉得刚毕业可以奋斗几年再去养老，人总是会活成自己讨厌的样子。一边享受着这舒适的生活，一边抱怨着它的不好，但是没有丝毫的改变，就像是温水的青蛙。其实非常的迷茫，想有所改变，但是又怀念这个舒适圈。偶尔也需要抬头看看天上的月亮 🌙。

简单来说，今年还是没有把当前的时间利用起来，懒癌也越来越严重，执行力逐步下降。明年也要想有所改变。

  • React 高级语法应用，源码知识的学习
  • React 后台系统的搭建
  • 数据结构/算法
  • 前端知识图谱的建立
  • Java 基础知识的重温
  • 计算机网络

生活&日常

做饭

疫情的缘故，在家买了一套厨具，春天在家上班的时间，几乎都会自己做饭，但是常做的还是那几样，排骨五花肉鸡翅鱼虾，还有常年不变的 xx 蛋汤。带饭去公司坚持了大概两个月的时间。天气冷了，做饭真的需要勇气，原来自来水是如此冻手的。
希望明年能够多尝试一些清淡的菜，能够把做饭这件事坚持下去。

看书

今年大概一些杂七杂八的书也就看了一两本，几乎闲暇的时间都在峡谷遨游了(也没上王者)。可以用手机 pad 打游戏，但是受不了看电子书，买 kindle 警告(不是)。
希望明年还是至少每月一本吧，12 本无关技术的书籍。

理财

今年的存钱目标也超额完成了，希望明年能达到你猜万。对于理财，是今年才开始接触的。从定期到债基到基金，一步一步接触风险更高的产品。年轻人要远离诺安银河招商生物，没想到人生的第一只鸡献给了诺安，从此走上了不归路，这该死的渣男，让人又爱又恨。今年行情很好，但是自己的韭菜操作，导致最后的收益率并不高，但也知足，没亏就是赚。
希望明年理财心态更好，减少韭菜操作，年化跑赢通货膨胀就行。也要好好挣钱/存钱，争取早日在北区上车。

兴趣

写下这个标题，我甚至不知道我有什么兴趣。突然发现自己是一个好无趣的人，周末就只会宅在家里躺尸。以前还偶尔出去拍照，今年相机吃灰一年吧。胶片的价格也越来越贵，拍的也少了，上次回卷还把胶片机弄坏了，洗出来的照片也不如人意，一定是拍照姿势不对。
希望明年也能好好拍照，杭州的四季都得有我！

其他

去年谈了恋爱，去年的总结也说了希望感情稳定，这大概是很好的完成了。一起成长，一起为以后的生活奔波。
回家日常被长辈念叨，日常电话也会被询问，考教资也要提上日程了。一次过。
若还没有被煮透，尝试再次回到互联网奋斗一把，做一些有成就感的事情。假装有这个事。

2020，有人见尘埃，有人见星辰，不过没关系，都要翻篇了。
2021，愿阳光满路上，温暖如初，继续奔走在自己的热爱里。
